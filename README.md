# SpringBootExam

## To Setup
###Order Services 
####Stand-alone
Edit application.properties .
####Docker
Edit docker-compose.yml .

###Notification Services 
####Stand-alone
Edit application.properties .
####Docker
Edit docker-compose.yml .

## To install:
###Order Services 
cd OrderServices  
docker-compose up .
###Notification Services 
cd NotificationServices  
docker-compose up .
